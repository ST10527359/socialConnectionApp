# Social Connection App ;)

## About
This app was built for IMAD5112 using Kotlin in Android Studio. 
The app is inspired by Cora, who struggles to stay connected with 
people during busy days. The app suggests small social actions 
based on the time of day to help her stay connected.

## Features
- Enter a time of day to get a personalised social spark suggestion
- Morning, Mid-morning, Afternoon, Afternoon Snack Time, Dinner and Night suggestions
- Reset button to clear the input and suggestion
- Error handling for invalid input

## How to Use
1. Type a time of day (e.g. Morning, Afternoon, Dinner)
2. Press "Get My Spark"
3. Read your suggestion
4. Press Reset to start again
